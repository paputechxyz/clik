Manage your LLM provider connection and view settings

Usage:
  myapp config [command]

Available Commands:
  llm         Connect an LLM provider (opencode / Claude / custom key)
  path        Print the config/settings file locations
  show        Show the resolved LLM provider (key redacted) and settings file

Flags:
  -h, --help   help for config

Global Flags:
      --db string   path to the SQLite DB file (default: ~/myapp/myapp.db or $MYAPP_DB_PATH)
      --json        emit machine-readable JSON (agent-native output)

Use "myapp config [command] --help" for more information about a command.
