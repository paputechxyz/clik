Runs an instant SQLite FTS5 full-text query over the fields of every stored
item — no network. Pass search terms as positional arguments (quoted if
multi-word). Use --exclude to drop unwanted terms.

Usage:
  myapp query <text> [flags]

Flags:
      --exclude strings   exclude terms (repeatable)
  -h, --help              help for query
      --limit int         max results (default 50)

Global Flags:
      --db string   path to the SQLite DB file (default: ~/myapp/myapp.db or $MYAPP_DB_PATH)
      --json        emit machine-readable JSON (agent-native output)
